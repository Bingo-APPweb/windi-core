# WINDI Memory Loop

Motor de inteligência fluídica. Memória institucional do ecossistema.

## Conceitos

### Osmose
Processo de absorção gradual de conhecimento entre sessões.
O Gêmeo não "lembra" — ele **absorve** padrões.

### Blocos de Sabedoria (Wisdom Blocks)
Unidades atômicas de conhecimento destilado:
- Decisões importantes
- Padrões que funcionam
- Erros a evitar
- Contexto institucional

### Destilação
Processo de extrair essência de conversas longas em blocos compactos.

## Ciclos Temporais

```
Sessão (minutos)     → Contexto imediato
Dia (horas)          → Wisdom Blocks do dia
Semana               → Padrões recorrentes
Mês                  → Conhecimento institucional
```

## Estrutura de Wisdom Block

```yaml
id: WB-2026-XXX
title: "Título descritivo"
context: "Situação que gerou o aprendizado"
insight: "O que foi aprendido"
aplicacao: "Quando/como aplicar"
tags: [tag1, tag2]
created: 2026-03-06
source: session_id
```

## Integração

Os Wisdom Blocks são armazenados em:
```
/opt/windi/wisdom/
├── WB-2026-001-*.md
├── WB-2026-002-*.md
└── index.json
```

## Princípio

> A memória não é armazenamento — é destilação.
> O que permanece é o que importa.
